# AdsBypasser - Domain Dashboard

A web dashboard for monitoring domains supported by [AdsBypasser](https://github.com/adsbypasser/adsbypasser).

[![Build Status](https://github.com/adsbypasser/domain-dashboard/actions/workflows/check-domains.yml/badge.svg)](https://github.com/adsbypasser/domain-dashboard/actions)

## Overview

Domain Dashboard provides a centralized interface to view, and track the status of domains that are handled by the AdsBypasser userscript. It helps contributors and maintainers keep track of which sites are supported, which need updates, and which are currently broken.

## Features

- View domain status (working, broken, needs update)
